# Expense Tracker (Python)

A simple GUI-based expense tracker built using Python and Tkinter.

## Features
- Add expenses
- View expenses
- Calculate total spending
- Data saved using JSON

## Technologies Used
- Python
- Tkinter
- JSON

## How to Run
1. Install Python
2. Run: `python main.py`
